Quick Installation

Get up and running with a single command

Install FilterDNS with this one-line command:

curl -sSL https://raw.githubusercontent.com/likhonsheikh54/FiltersDNS/main/dns.sh | sudo bash

Or download and run the script manually:

wget https://raw.githubusercontent.com/likhonsheikh54/FiltersDNS/main/dns.sh
chmod +x dns.sh
sudo ./dns.sh

After installation completes:

  1. Access the web interface at http://YOUR_SERVER_IP
  2. Configure your network devices to use your server's IP as the DNS server
  3. Test the filtering by visiting http://YOUR_SERVER_IP/test.html

How to Use

FilterDNS is easy to manage through both web interface and command line

Web Interface

The web interface allows you to:

  • Monitor server status and statistics
  • Enable/disable filtering categories
  • View top queried and blocked domains
  • Start, stop, or restart the DNS server
  • Update blocklists manually

Access at: http://YOUR_SERVER_IP

Command Line

Control FilterDNS from the command line:

sudo /opt/filterdns/bin/dns-control.sh [command]

Available commands:

  • status: Show DNS server status
  • start: Start DNS server
  • stop: Stop DNS server
  • restart: Restart DNS server
  • update: Update blocklists
  • enable [category]: Enable a category
  • disable [category]: Disable a category
  • stats: Show DNS server statistics

Frequently Asked Questions

Common questions about FilterDNS

What is DNS filtering?

DNS filtering blocks access to specific websites or content by preventing DNS resolution of domain names. When a user tries to access a blocked domain, the DNS server returns an error or redirects to a block page instead of the actual IP address.

What content categories can be blocked?

FilterDNS includes blocklists for adult content, malware, ads, and tracking domains. You can enable or disable each category independently, and add custom domains to block or whitelist.

How do I add custom domains to block?

You can add custom domains to block by editing the file at /opt/filterdns/data/blocklists/custom.txt and then running sudo /opt/filterdns/bin/dns-control.sh update to apply the changes.

Can I use FilterDNS on my home network?

Yes! FilterDNS is perfect for home networks. Install it on any Ubuntu server or virtual machine, then configure your router to use it as the DNS server for your entire network.

Is FilterDNS suitable for business use?

Absolutely. FilterDNS can be deployed in business environments to enforce content policies, block malware, and protect your network. For production environments, we recommend setting up SSL/TLS with Let's Encrypt for the web interface.
